Functions:

*** It has a keyword as "function".
SYNTAX:
1. Definition of the function.
function name_of_the_function(parameters/arguments)
{
    //body of the function.
}
2. Function calling.
name_of_the_function(parameters/arguments).

*** Return Statement:
    ---> If a function doesn't return any value, a default value is returned which is the value "undefined".

Ways of defining/calling a function...
1. Named functions.
2. Anonymous functions (not having any name).
3. Self Invoking functions (func calling itself).
4. Constructors.
