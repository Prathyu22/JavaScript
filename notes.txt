OBJECT CREATION: (object---logical entity)
1.Creating object with NEW keyword.
       var var_name = new Object();
       var_name.property_name = "___";
       var_name.property_name = "___";
2.Creating {} with initial set of properties (attributes) with .(dot)
       var var_name = {};
       var_name.property_name = "___";
       var_name.property_name = "___";
3.Creating initial set of properties(attributes) with []
       var var_name = {};
       var_name['property_name'] = "___";
4. Creating with initial set of properties(attributes) with {}
       var var_name = { property_name: "___", property_name2: "___"};

5. "THIS" keyword in object creation using constructor function:
function func_name() {
    this.property_name = "____";
    this.property_name = _____;
}
var var_name = new func_name();
